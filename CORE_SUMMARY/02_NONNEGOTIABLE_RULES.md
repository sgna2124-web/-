# NONNEGOTIABLE RULES

## 절대 기준
1. 과거의 좋아 보이는 숫자를 맹신하지 않는다.
2. 현재 재현 가능한 코드와 결과만 공식 기준으로 삼는다.
3. 단일 종목 검증과 전체 종목 재검증이 맞아떨어져야 한다.
4. 실전 기준을 우선한다.

## 엔진/백테스트 관련 고정 규칙
- same-bar revalidated 엔진 기준을 우선한다.
- fee_per_side = 0.0004 확인.
- 동시 포지션 수 기반 자산 분할 원칙을 유지한다.
- 사용자가 지정한 시간축/합성 규칙을 빼먹지 않는다.
- 기존 기준선과 비교할 때는 동일한 엔진/동일한 비용 가정/동일한 자산 분할 전제를 맞춘다.

## 금지수칙
- pre-v22 계열 과거 기준을 공식 기준선처럼 재사용하지 말 것.
- 검증되지 않은 고수익 수치를 근거로 전략 우수성을 주장하지 말 것.
- topN/top200 계열 규칙을 이론적 정답처럼 고정하지 말 것.
- 버그가 있었던 original workflow 계열을 무심코 재사용하지 말 것.
- 결과가 좋아 보여도 재현성 확인 전에는 승자 확정 금지.

## 문서화 규칙
새 전략/실험을 추가할 때 반드시 아래를 남긴다.
- 왜 이 전략을 시도했는가
- 이전 기준선 대비 무엇을 바꿨는가
- 기대한 개선 포인트는 무엇이었는가
- 결과는 어땠는가
- 왜 유지/폐기/보류하는가

## 운영 태도
- 전략 개선보다 전략 복원/정의가 먼저 필요한 경우에는 그 사실을 명시한다.
- 수익률만 보고 흥분하지 않는다.
- MDD, max_conc, 실전 가능성까지 본다.
