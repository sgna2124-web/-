로컬 파이썬 백테스트 환경 규칙

파이썬 백테스트 환경은 깃허브 저장소 환경과 다르며, 저장소 내부 파일이나 경로를 참조하는 것은 불가능하다.

적용 원칙
1. 로컬에서 실행될 백테스트 .py 파일은 실행 중 깃허브 저장소 내부 경로를 import, open, read_json, glob 방식으로 참조하면 안 된다.
2. 기준선 전략 조건, 참조값, 조합 설정이 필요하면 코드 생성 시점에 확인해 단일 실행 파일 내부에 직접 탑재한다.
3. 사용자의 로컬 실행 파일은 외부 저장소 파일 접근을 전제로 만들지 않는다.
4. 결과 대조용 코드도 저장소 결과 파일을 로컬에서 찾게 만들지 않는다. 필요한 참조값은 파일 내부에 내장한다.
5. 저장소 파일을 수정하거나 확인하는 작업은 어시스턴트가 GitHub 도구로 수행하고, 사용자의 로컬 파이썬 실행 과정에는 의존시키지 않는다.
