# 테스트7

이 폴더는 자동매매 전략 개발 프로젝트의 현재 대화 기준 인수인계/복기용 기록 저장소다.
목적은 사람이 읽기 좋게 만드는 것이 아니라, 다음 대화의 어시스턴트가 이전 실수와 현재 상태를 빠르게 복원하도록 만드는 것이다.

구성
- 00_INDEX.md: 전체 구조와 읽기 순서
- 01_RULES_AND_ENGINE.md: 프로젝트 목표, 평가 철학, 엔진 제약, 금지 조건
- 02_DATA_AND_HISTORY.md: 데이터 범위, 사용자 요구 변화, 실패 축, 시도 이력
- 03_VERIFIED_RESULTS.md: 대화 중 실제 수치로 명시된 백테스트 결과
- 04_STATE_REFRAME_AND_PINE.md: 상태 기반 재구성, PineScript 수정 이력, 최신 문제점
- 05_NEXT_PLAYBOOK.md: 다음 세션에서 바로 실행할 순서와 주의사항

최우선 읽기 순서
1. 01_RULES_AND_ENGINE.md
2. 03_VERIFIED_RESULTS.md
3. 04_STATE_REFRAME_AND_PINE.md
4. 05_NEXT_PLAYBOOK.md

사용 원칙
- 수치가 있어도 재현 가능한 정의가 없으면 최종 진실로 취급하지 말 것
- 부분 샘플 결과를 전체 결과처럼 쓰지 말 것
- 사용자가 업로드를 계속 추가하므로 다음 세션에서 파일 목록, 중복, 기간은 반드시 다시 집계할 것
- 이 폴더는 무엇이 이미 실패했는지 기억하는 용도가 특히 중요하다
