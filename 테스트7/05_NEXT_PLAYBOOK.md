1. 프로젝트 규칙 재확인
- OHLCV only
- 5분봉 only
- 포트폴리오 순차 시뮬
- 진입당 자산 1퍼센트 고정
- 동일봉 SL 우선
- 수수료 왕복 0.08퍼센트
- TP 예상 수익폭 0.3퍼센트 미만 진입 금지
- 재현성 검증

주의
- 사용자는 후반 Pine 실험에서는 유동 SL TP를 원했지만, 원래 공식 백테스트 엔진 규칙은 고정 SL TP only였다.
- 다음 세션에서는 공식 엔진 규칙을 유지할지, 새 실험 축으로 유동 SL TP를 별도 브랜치화할지 내부적으로 정리하고 시작할 것.

2. 상태 분류 프레임
- 추세: 상승, 하락, 횡보
- 변동성: 높음, 중간, 낮음
- 최근 대비 상대 변화 기준
- 애매한 상태는 거래 금지
- 방향성 없는 고변동 상태는 거래 금지
- 상승 상태 내부 세분: 완만한 눌림, 급락 후 반전 후보
- 횡보 내부 세분: 저변동 박스, 방향성 없는 고변동 난조

3. 전략 분리 원칙
- 상승 상태: 하락 후 초기 반전 롱
- 하락 상태: 상승 후 초기 반전 숏
- 횡보 상태: 상단 반전 숏, 하단 반전 롱
- 롱과 숏은 대칭으로 뒤집지 말고 별도 설계

4. 우선순위
1) 상승 상태 전용 롱 구조 정교화
2) 하락 상태 전용 숏
3) 횡보 상태 반전

5. 진입 위치에 대한 강한 제약
- 하락 중 칼날 잡기 금지
- 고가 돌파 후 과도하게 늦은 진입 금지
- 목표 위치는 하락 종료 + 초기 반전 시그널
