돈버는 차트 전략 프로젝트 - TEST1 체크포인트

최종 목표
- 자동매매로 장기적으로 돈을 버는 전략 개발
- 5분봉 기반
- 바이낸스 선물 전체 종목 대상
- 멘탈 고려 없음
- 고빈도/저빈도보다 EV가 중요

핵심 철학
- 패턴, 추세, 구조를 믿지 않는다.
- 상태 -> 확률 -> 기대값(EV) 구조만 유지한다.
- 승률 자체보다 비용 포함 EV가 플러스인지가 핵심이다.

운영 제약
- 실전에서는 각 심볼마다 최근 1500개 OHLCV만 조회 가능하다고 가정한다.
- 따라서 모델은 장기 저장형이 아니라 최근 1500개로 계속 재평가하는 형태를 우선 연구한다.

절대 금지
- 다시 RSI/망치/전통 패턴 해석 중심 전략으로 회귀
- 조건을 계속 덧붙여 거래가 사라지는 과적합 구조
- 토큰, API 키, 시크릿, 개인 정보 저장

개발 원칙
- 기존 틀(엑셀 저장, print, plt, 순차거래 시뮬)은 유지 가능해야 한다.
- 진입 조건만 독립적으로 교체 가능한 구조가 좋다.
- 실험 로그를 남겨 같은 실수를 반복하지 않는다.
