# 06_USER_INTENT_AND_COMMUNICATION_RULES

- 사용자는 목표를 숨기지 않는다. 목표는 돈을 버는 전략이다.
- 용어만 바꾸는 말장난을 매우 싫어한다.
- 같은 질문 반복, 같은 전략축 반복, 이미 실패한 흐름 반복에 매우 민감하다.
- 포기 여부를 계속 묻는 것도 매우 싫어한다.
- 반드시 기존 엔진 틀 기준으로 말해야 한다.
- 결과는 승률보다 최종 순차 수익과 MDD 중심으로 설명해야 한다.
- 세션/시간/압축폭발/OI/구조 재제안 금지.
- 타임프레임 키우면 해결된다는 식의 단순 제안 금지.
- 단일 종목 미니 코드 제공 금지.
