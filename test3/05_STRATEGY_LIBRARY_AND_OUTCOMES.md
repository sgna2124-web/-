# 05_STRATEGY_LIBRARY_AND_OUTCOMES

사용자가 실제로 의미 있다고 느낀 전략 철학

1. 볼린저 복귀 전략
- 볼린저밴드 사용
- 소스는 시가
- std 7
- 종가가 밴드 밖으로 이탈 후 복귀하면 역추세 진입
- 평가: MDD는 높지만 수익 기대치가 꽤 높았음
- 사용자가 중요하게 본 점: 비정상 상태, 짧은 SL, 선명한 무효화, 명확한 TP

2. Failed Displacement Reclaim (새 철학으로는 상대적으로 양호)
- 최근 수용 범위를 비정상적으로 강하게 이탈
- 그러나 새 가격대 수용 실패 후 다시 range 안으로 reclaim
- 역추세 진입
- 평가: 철학은 괜찮다고 봤으나 초기 테스트 성과는 실패

폐기된 전략 카드들
- TrendPullback 계열
- Market Shock Propagation
- Cross-sectional isolated reversion
- Minor Washout Reversal v1/v2
- Symbol Fit / Symbol State Washout 계열
- 사람이 이름 붙인 이벤트 전략 전반

중요 결론
- 사용자는 정상 상태의 세밀한 우위보다, 사건 중심의 비정상 상태 전략을 더 신뢰한다
- 단, 철학이 좋다고 성과가 보장되는 것은 아니므로 기존 엔진 기준으로 실제 검증이 필요하다
