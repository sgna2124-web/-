# 00_PROJECT_MANUAL

프로젝트 이름: 돈버는 차트 전략 찾기

사용자 구분
- 코딩/거래 전략/차트/코인 시장 질문자는 '부자창대'
- 그 외 일반 질문자는 '로박사'
- 이 프로젝트 문맥에서는 반드시 사용자를 부자창대로 인식

최종 목표
- 자동매매로 장기적으로 돈을 버는 전략 만들기
- 5분봉 기반
- 바이낸스 선물 다종목 동시 운용
- 인간 멘탈 고려 없음
- 승률 자체가 목표가 아님
- RR 자체가 목표가 아님
- 최종 기대값(EV)이 플러스면 됨

핵심 철학
- 카지노처럼 아주 작은 통계적 우위라도 반복해서 돈을 버는 구조를 원함
- 하지만 '평범한 상태의 미세 우위'보다, 비정상 상태 + 선명한 무효화 + 짧은 SL + 분명한 TP를 더 선호함
- 말장난이나 용어 변경은 의미 없음. 결국 돈 버는 전략이어야 함

강한 고정 제약
- 5분봉 유지
- OHLCV만 사용
- 고정 SL/TP
- 추가 청산 없음
- 순차 포트폴리오 시뮬레이션 기준 평가
- 기존 백테스트 엔진 틀 유지

가장 중요한 경고
- 같은 실패 축 반복 금지
- 세션/시간/압축폭발/구조/OI/일반 추세/일반 패턴/미니 테스트 코드 재제안 금지
- 사용자는 이미 많은 전략을 실제로 백테스트했고, 비슷한 제안을 매우 싫어함

현재 프로젝트 상태
- 5분 OHLCV-only 공간에서 일반적 차트형/상태형/이벤트형 탐색은 거의 소진됨
- state_miner 결과가 0이면 5분 OHLCV-only 공간 종료를 공식 선언할 의향이 있음
- strict survivor가 나오면 즉시 기존 엔진에 삽입해 전략 테스트할 것
