# 01_FIXED_CONSTRAINTS

절대 고정 제약
- 데이터: OHLCV만 사용
- 타임프레임: 5분봉
- 시장: 바이낸스 선물 다종목
- 청산: 고정 SL/TP만 사용
- 추가 청산, BE, 트레일링, 조건부 청산 금지
- 평가: 순차 포트폴리오 시뮬레이션 기준

사용자 관점
- 5분에서 실패한 미시구조 전략이 더 큰 타임프레임으로 간다고 성공한다고 보지 않음
- 수수료 핑계 금지: 평균 변동성 0.3% 이상 등으로 이미 필터를 고려했음
- 정상 상태를 잘게 나누는 미세 우위보다, 비정상 상태에서 명확한 무효화와 짧은 SL을 선호

실제 운영 제약
- 최근 1500개 캔들 요청 환경을 기준으로 판단하는 것이 자연스러움
- 하지만 1500개를 반드시 다 써야 하는 것은 아니고, 더 적은 캔들로 상태를 판정할 수 있으면 더 좋음

평가 핵심
- 승률은 부차적
- 최종 순차 수익, MDD, 최대 수익/손실 종목, 최대 포지션 보유 수가 더 중요
