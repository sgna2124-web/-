# 02_FAILURE_AXES_DO_NOT_REPEAT

아래 축은 이미 충분히 시도했고 실패했거나, 다시 반복할 가치가 없다고 판단된 영역이다.

재제안 금지 목록
- 추세추종 전략
- 구조 기반 전략
- 시장 상태 분류 엔진
- 변동성 압축 돌파
- 패턴 기반 전략
- adaptive TP/SL
- fractal pullback
- 세션/시간/요일 효과
- 세션을 변동성으로 해석하는 반복 접근
- 단일 종목 이벤트 이름 기반 접근
- OI 활용 전략
- 고립형 평균회귀
- 시장 충격 전파
- minor washout 계열의 여러 변형
- 사람이 이름 붙인 OHLCV 이벤트 스캐너 접근
- 넓은 상태공간을 일반적으로 훑으며 '평범한 상태의 미세 우위'를 찾는 접근
- 타임프레임 상향 도피성 제안
- 기존 실패 전략을 조금 손보는 식의 반복

중요한 이유
- 사용자는 같은 실수, 같은 과정, 같은 시간 낭비를 매우 싫어함
- 같은 흐름을 다시 반복하면 신뢰를 잃음
- 새 아이디어를 제안하기 전에 반드시 이 목록과 충돌하는지 먼저 확인해야 함
