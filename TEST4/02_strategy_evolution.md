# 전략 진화 기록

초기 단계
- Immediate Climax Reversal 계열 템플릿 사용
- 비정상 봉 단독 반전 포착 시도
- 이벤트 직후 반응 포착은 실패

중기 통찰
- 이벤트 이후 10~20봉 흡수 존재
- 살아남은 구조는 비정상 확장 -> 전달 실패 -> 복귀
- 숏 쪽이 롱보다 일관적으로 강함

탐색 흐름
- E1~E16 숏 계열, L1~L10 롱 계열로 내부 후보를 늘려가며 비교
- 데이터 추가 시 종목 수가 늘어날수록 성과가 좋아지는 구간이 있었으나, 과적합 가능성 높다고 판단
- 종목별 편차가 존재하여 향후 기간/종목 분할 검증 필요
