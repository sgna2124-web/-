# TEST4 프로젝트 운영 매뉴얼

프로젝트명: 자동매매 전략 개발2

핵심 목표
- 자동매매로 장기적으로 돈 버는 전략
- 평가지표는 EV 우선
- 최종 판단은 포트폴리오 순차 시뮬 기준

필수 제약
- OHLCV only
- 5분봉 only
- 고정 SL/TP only
- 수수료 편도 0.04%
- TP 예상폭 0.3% 미만 진입 금지
- 각 진입은 진입 시점 equity의 1%만 사용
- 동일봉 SL/TP 동시 터치 시 SL 우선
- 재현성 필수

핵심 구조
- 비정상 확장 -> 전달 실패 -> 복귀
- 숏: 강한 상방 확장 실패 후 돌파 레벨 하향 복귀
- 롱: 강한 하방 확장 실패 후 이탈 레벨 상향 복귀

사용자 금지사항
- 같은 질문 반복 금지
- 실패한 축 재제안 금지
- 중간 결과 보고 금지
- 장난감 코드 금지
- 말만 많은 설명 금지

작업 루프
1. 원시 사건 재스캔
2. 상위 사건 전략화
3. 백테스트
4. 엔진 규칙 검증
5. 성과 미달 시 폐기 후 1번 복귀
6. 성과 전략만 변형
7. 최종 수익률 > MDD 전략만 통과
