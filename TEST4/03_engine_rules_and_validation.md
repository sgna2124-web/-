# 엔진 규칙 및 검증 체크리스트

불변조건
- position_notional = entry_equity_snapshot * 0.01
- 포지션별 size는 진입 시점에 고정
- 단일 트레이드 손익은 논리적 RR 범위를 벗어나면 즉시 폐기
- 동일 입력 2회 이상 실행 시 완전 동일 결과여야 함
- Peak / MDD 계산 일관성 유지
- NaN / inf / 비정상 값 발생 시 결과 폐기

검증 순서
1. 전략 백테스트 1차 실행
2. 1% 규칙 검사
3. RR 범위 검사
4. 동일봉 SL 우선 처리 검사
5. 재실행 후 재현성 비교
6. 통과 전략만 보고
