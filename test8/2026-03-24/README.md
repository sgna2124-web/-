# TEST8 작업 인덱스

목적: 다음 대화창의 내가 바로 이어서 작업할 수 있도록, 이 프로젝트의 목표/검증 규칙/전략 탐색 기록/현재 기준 전략/실수 방지 규칙을 작업자 기준으로 정리한다.

## 현재 프로젝트 핵심 목표
- 목표는 '그럴듯한 전략'이 아니라 '실제로 돈 버는 전략'이다.
- OHLCV만으로 전략을 만든다.
- 실전 운용 가능한 백테스트만 인정한다.
- 롱/숏 모두 허용한다. 한쪽 편향으로 고정하지 않는다.
- 자산 1% 진입 기준 결과를 먼저 보고, 이후 최대 동시 포지션 수를 바탕으로 재분할 재검증한다.
- 수수료는 진입/청산 각각 0.04%, 왕복 0.08% 반영이 기본이다.
- 백테스트는 전체 종목 기준으로 본다. 일부 샘플 결과만으로 결론 내리지 않는다.

## 현재 확정된 기준 전략
전략명: 과확장 복귀 전략 v1

핵심 성과(자산 1% 진입 기준)
- 최종자산: 1.572
- PF: 1.098
- 승률: 40.09%
- MDD: -11.95%
- 최대 동시포지션: 192

이 전략은 반드시 기억한다. 이후 새 전략을 검증할 때 비교 기준선으로 사용한다.

## 작업 메모
- 이전 전략들을 맹목적으로 답습할 필요는 없다.
- 목표에 맞으면 새 전략을 직접 만들고 백테스트해서 채택하면 된다.
- 사용자는 결과가 아니라 재현성과 실전성을 더 중요하게 본다.
- 그러나 최종 목표는 결국 돈 버는 전략이다. 수익 없는 전략은 빠르게 폐기한다.
