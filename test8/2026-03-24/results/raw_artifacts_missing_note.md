# 원시 산출물 누락 사유

## 왜 이 파일이 필요한가
이번 테스트8 업로드는 문서뿐 아니라 재현용 자산까지 채우는 것이 목적이었다. 다만 세션 중 일부 백테스트는 대화형 실행 환경에서 즉시 돌리고 숫자만 확인한 뒤, 별도의 csv/json/raw trade log로 저장하지 못한 상태에서 세션이 끝났다.

## 현재 저장 가능한 것
- 전략 설명 문서
- 기준 전략 파라미터 json
- 벤치마크 결과 json
- 재분할 결과 json
- 전략 배치 결과 요약 json
- 다음 탐색 큐와 프로토콜 문서

## 현재 없는 것
- 전략별 전체 raw trades csv
- 종목별 상세 성과 csv
- 세션 중간 python notebook 상태 그대로의 실행 스냅샷
- 5개 대기 후보(squeeze breakout 등)의 완주 결과 파일

## 이유
- 실행 환경 세션 리셋/중단 발생
- 일부 실험은 대화 중 숫자 확인 후 바로 전략 판정만 남기고 파일 저장을 하지 않음
- 따라서 지금 테스트8에는 '재구성된 결과 파일'과 '확정된 수치 메모'를 우선 보관함

## 다음 작업 지침
- 다음 세션 시작 시 기준 전략 과확장 복귀 전략 v1부터 파일로 재현
- raw trades / per-symbol summary / aggregate summary를 모두 파일로 저장
- 이후 신규 전략 배치도 각 전략별 json/csv로 즉시 저장
