# 반복 금지 실수 목록

## 사용자 관점에서 이미 크게 문제 되었던 것
- 실수한 백테스트를 마치 확정 결과처럼 말하는 것
- 일부 종목/일부 샘플만 보고 전체 전략처럼 일반화하는 것
- 숏만 혹은 롱만 고집해서 목표를 잘못 해석하는 것
- 실패한 전략을 오래 붙잡고 시간 낭비하는 것
- 결과가 안 좋을 때 바로 다음 전략으로 넘어가지 않는 것

## 구현/검증 관점 금지 사항
- 룩어헤드
- 신호봉과 진입봉 개념 섞기
- 수수료 누락
- 전체 종목 대신 일부 샘플만으로 결론 내리기
- 기대 TP가 낮은데도 무작정 진입시키기
- 최대 동시포지션을 무시한 채 성과 과대평가하기

## 전략 탐색에서 이미 확인된 실패 패턴
- 단순 돌파
- 단순 반전
- 단순 EMA 크로스
- 구조 없는 RSI 역추세
- 거래 수가 비정상적으로 많은 반전 전략

## 작업 태도 메모
- 사용자는 장황한 설명보다 핵심 수치와 판정을 원함
- 질문으로 시간을 끌지 말고 직접 시도해야 함
- 실패하면 바로 스스로 피드백하고 다음 전략으로 이동해야 함
