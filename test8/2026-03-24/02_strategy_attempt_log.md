# 전략 시도 로그

아래 로그는 이번 구간에서 확인된 전략 탐색 흐름을 정리한 것이다. 일부 수치는 대화 중 확정된 값 위주로 정리했으며, 실행 도구 오류로 완전한 로그 파일이 남지 않은 구간은 서술형으로 남긴다.

## 폐기된 큰 방향
1. 단순 RSI 반전 + 꼬리
- 롱만 혹은 롱숏 대칭으로 테스트
- 구조가 약하고 수수료에 짓눌림
- 사실상 수익 없음
- 판정: 폐기

2. 단순 추세 + 돌파
- MA/EMA 상승 상태에서 단기 고점 돌파 롱, 하락 상태에서 저점 이탈 숏
- 이미 움직인 뒤 진입해서 되돌림에 취약
- 판정: 폐기

3. 돌파 실패 반전 롱숏
- 20봉 고점/저점 돌파 시도 후 종가 복귀 + 꼬리 우세
- 전체 종목 기준 거래가 과도하게 많아짐
- 기록된 결과 예시:
  - 거래수: 463,337
  - 최종자산: 0.0359
  - 승률: 35.64%
  - PF: 0.805
  - MDD: -96.41%
  - 최대 동시포지션: 148
- 판정: 완전 폐기

4. 추세 돌파 + 거래량 / 눌림목 / EMA 크로스 등
- 여러 계열을 묶어서 탐색
- 공통적으로 수수료와 군집 손실에 취약
- 판정: 폐기

## 비교적 덜 나빴으나 폐기된 방향
1. 극단 꼬리 + RSI 반전 롱숏
- 덜 나쁘지만 우위가 약함
- 기록 메모상 최종자산 0.742 수준, PF 0.914, MDD -27.6%, 최대 동시포지션 87
- 판정: 폐기

2. EMA 이격 과확장 평균회귀 롱숏 (초기 버전)
- TP 0.3% 기대값 필터를 강화하기 전/후 여러 변형 존재
- 일부 버전은 거의 손익분기 수준
- 이 라인을 다듬은 결과가 현재 벤치마크인 과확장 복귀 전략 v1으로 이어짐

## TP 기대값 0.3% 이상 필터 추가 이후
여러 전략을 재탐색했고, 그중 기록이 남은 핵심 결과는 아래와 같다.

1. 돌파 실패 반전 롱숏
- 최종자산 0.329
- PF 0.834
- MDD -67.1%
- 최대 동시포지션 97
- 판정: 폐기

2. EMA 이격 과확장 평균회귀 롱숏
- 최종자산 0.947
- PF 1.003
- MDD -35.3%
- 최대 동시포지션 148
- 판정: 경계선, 사실상 폐기

3. 압축 후 확장 돌파 롱숏
- 최종자산 0.947
- PF 0.864
- MDD -6.1%
- 최대 동시포지션 17
- 판정: 손실 전략, 폐기

4. 추세 눌림목 복귀 롱숏
- 최종자산 0.031
- PF 0.879
- MDD -97.3%
- 최대 동시포지션 178
- 판정: 완전 폐기

5. 과확장 후 평균회귀 롱숏 → 최종 벤치마크 채택
- 이 라인을 발전시켜 과확장 복귀 전략 v1 확정

## 아직 제대로 끝까지 평가 못한 대기 후보
- squeeze breakout
- inside bar trend
- base breakout
- retest breakout
- pullback trend

주의: 위 대기 후보는 '우선순위 큐'일 뿐 채택된 전략이 아니다.
