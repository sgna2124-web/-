통합요약1

목적
이 폴더는 test1 ~ testN에 흩어진 전략 실험 기록을 다음 대화창에서 빠르게 재사용하기 위한 기준서다.
상세 원문은 각 test 폴더에 남기고, 여기에는 지금도 유효한 핵심 판단만 남긴다.

2. 01_전략통합요약.md

4. 03_장점추출_아이디어박스.md
5. 05_업데이트규칙.md

구조
- 01_전략통합요약.md : 큰 그림
- 01A_롱전략계통.md : 롱 상세
- 01B_숏전략계통.md : 숏 상세

- 02B_전략설계_반복금지.md : 전략 측면의 반복 금지
- 02C_평가기준_혼용금지.md : 순위 기준, 비교 방식
- 03_장점추출_아이디어박스.md : 재사용 가능한 장점
- 04_현재기준_TOP1_참조.md : 현재 롱 1위, 숏 1위, 비교 기준 수치

원칙
- test 폴더는 원본 상세 기록이다.
- 통합요약1은 기준서다.
- 새 테스트가 끝나면 test 폴더에 상세 기록을 남기고, 여기에는 바뀐 판단만 반영한다.
- 파일 하나에 모든 내용을 누적하지 말고, 의미 단위별 하위 파일로 분산한다.
