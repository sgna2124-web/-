반복금지 체크리스트

실행/저장소
- LFS pointer 상태의 데이터를 실제 데이터로 착각하고 실행하지 말 것
- 저장소 쓰기 실패를 내용 문제로 오판하지 말 것
- 무거운 다중 파일 쓰기를 한 번에 밀어 넣지 말 것

평가기준
- 로컬 실험 1위와 저장소 전체 1위를 혼동하지 말 것
- 결과 보고 시 여러 개를 늘어놓지 말고 현재 기준 1위만 먼저 말할 것
- 순위 기준은 MDD 5% 미만 후보 중 cd_value 1위로 통일할 것

전략설계
- 파라미터만 미세하게 흔드는 반복을 하지 말 것
- 이미 포화된 축을 큰 이유 없이 다시 반복하지 말 것
- 숏에서 잘 된 구조를 롱에 무비판적으로 복제하지 말 것
- 메인형과 공격형을 같은 잣대로 섞지 말 것
- 극저 MDD만 보고 롱을 좋다고 판단하지 말 것
