# TEST6 README

이 폴더는 돈버는 차트 전략 찾기 프로젝트의 TEST6 인수인계용 기록이다.
목적은 이전 대화에서 발생한 착시, 구현 오류, 기준 혼선을 반복하지 않고, 다음 작업자가 즉시 현재 상태를 이해하게 만드는 것이다.

핵심 결론:
- 과거 대화에서 나온 고수익 수치(+263%, +252%, +1187% 등)는 현재 시점에서 신뢰 가능한 기준이 아니다.
- 현재는 "재현 가능한 코드 + 단일 종목 검증 + 전체 종목 재검증"만 기준으로 삼아야 한다.
- 현재 디버그 구현은 KAIA/USDT에서는 1회 진입, CGPT/USDT에서는 0회 진입이 재현된다.
- 전체 종목 재검증 결과, 현재 구현 전략은 수익 전략으로 확정할 수 없다.
- 즉 지금 단계는 전략 개선보다 전략 복원/재정의에 가깝다.

폴더 구성:
- 00_README.md: 전체 개요
- 01_user_requirements.md: 사용자 요구사항/금지사항/운용 기준
- 02_strategy_timeline.md: 전략 탐색 흐름과 의사결정 타임라인
- 03_postmortem_false_positive_results.md: 왜 긍정적 결과가 나왔는지, 왜 폐기하는지
- 04_current_verified_state.md: 현재 신뢰 가능한 상태와 수치
- 05_python_implementation_notes.md: 파이썬 구현상 핵심 로직/오류/수정 포인트
- 06_pinescript_and_backtest_rules.md: 파인스크립트 주의사항 + 실전 백테스트 원칙
- 07_next_steps.md: 다음 작업 우선순위
- 08_strategy_debug_5m_synth_1h.py: 현재 디버그 기준 파이썬 코드 스냅샷

이 폴더를 읽을 때의 원칙:
1. 과거의 좋아 보이는 숫자를 기준점으로 삼지 말 것.
2. 지금 재현되는 결과만 기준으로 판단할 것.
3. 단일 종목 검증과 전체 종목 결과가 맞아떨어지는지 먼저 볼 것.
4. 사용자가 요구한 "실전 기준", "동시 포지션 수 기반 자산 분할", "5분봉으로 합성한 1시간봉"을 절대 빼먹지 말 것.
