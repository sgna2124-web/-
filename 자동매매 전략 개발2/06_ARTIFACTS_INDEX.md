# 06_ARTIFACTS_INDEX

이 폴더/저장소에는 이번 대화에서 생성된 핵심 산출물만 업로드한다. 용량이 과도한 대형 트레이드 로그는 제외하거나 요약본만 둔다.

## CSV / 결과 파일
- results/strategy20_results.csv: 초기 20개 전략 결과 요약
- results/new_strategy20_results.csv: 다른 결의 20개 전략 결과 요약
- results/new_strategy20_symbol_results.csv: 위 전략들의 종목별 성과
- results/new_strategy20_symbol_bestworst.csv: 종목별 최고/최악 전략
- results/raw_event30_stats.csv: 30개 원시 사건 전체 통계
- results/raw_event30_symbol_stats.csv: 30개 원시 사건 종목별 통계
- results/top3_strategy_results.csv: 상위 3개 전략 테스트 결과
- results/top3_strategy_symbol_results.csv: 상위 3개 전략 종목별 결과
- results/nr5f_variant_fastscan.csv: NR5-F 주변 빠른 변형 스캔
- results/nr5f_best2_results.csv: NR5-F 상위 2개 전략 결과
- results/nr5f_best2_symbol_results.csv: NR5-F 상위 2개 종목별 결과
- results/rescan46_event_stats.csv: 46개 재스캔 전체 통계
- results/rescan46_top_candidates.csv: 46개 재스캔 상위 후보
- results/clx5_variant_results.csv: CLX 주변 5개 변형 결과
- results/clx5_variant_symbol_results.csv: CLX 주변 변형 종목별 결과
- results/rescan50_event_stats_actual.csv: 50개 재스캔 전체 통계
- results/rescan50_top20_actual.csv: 50개 재스캔 상위 20개
- results/rescan50_top5_backtest_summary.csv: 상위 5개 전략화 결과
- results/rescan50_top5_backtest_symbol.csv: 상위 5개 종목별 결과
- results/three_strategy_backtest_results_actual.csv: 3개 전략 실제 백테스트 요약
- results/three_strategy_backtest_symbol_results_actual.csv: 3개 전략 종목별 결과
- results/volrev_improve_summary.csv: VOLREV 개선안 요약
- results/volrev_improve_symbol.csv: VOLREV 개선안 종목별 결과
- results/noextend_variant_backtest.csv: noextend 변형 5개 결과
- results/noextend_rr_event_grid_results.csv: noextend RR x 이벤트강도 그리드
- results/noextend_best_symbol_results.csv: noextend 최적 조합 종목별 결과
- results/acc3_variant5_actual.csv: ACC3 변형 5개 결과

## 주의
- 일부 대형 trade log(csv)는 크기가 너무 커서 저장소에 그대로 올리지 않음
- 로그 전체가 필요하면 향후 별도 압축 업로드 필요
- 그러나 현재 의사결정에는 요약 결과와 규칙 문서가 더 중요함
