# 01_NON_NEGOTIABLES

## 절대 제약
- 기본 타임프레임은 5분봉
- 15분/1시간/상위 타임프레임 제안 금지
- OHLCV only
- OI는 이미 실패했으므로 다시 사용 금지
- 실제 평가는 개별 심볼이 아니라 전체 포트폴리오 시간순 순차 시뮬 기준

## 작업 규칙
- 장난감 백테스트 코드 금지
- 단일 종목 미니 테스트 코드 금지
- 승률만 출력하는 코드 금지
- 폴더 전체 xlsx 무차별 로드 금지
- 반드시 기존 엔진 틀 또는 단일 고정 엔진 구조 안에서만 전략 로직 교체

## 사용자 응답 선호
- 쓸데없는 설명 싫어함
- 같은 질문 반복 싫어함
- 실패한 전략축 재제안 싫어함
- 결과 위주, 빠른 실행 선호
- 중간 실패 보고보다 실제 카드가 나왔을 때만 의미 있는 보고 선호

## 고정 루프
원시 사건 재스캔 -> 상위 후보 전략화 -> 백테스트 -> 좋은 전략만 변형 5개 -> 안 좋으면 재스캔
